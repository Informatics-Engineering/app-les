## Require
1. Composer
2. NPM
3. Mysql Database
4. PHP 7.4+
## Instalasion
1. Clone this repository
2. configure .env file
3. Run `Composer install`
4. Run `php artisan key:generate`
7. Run `php artisan migrate`
8. Run `php artisan serve`
